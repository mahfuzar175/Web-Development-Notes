CSS UNITS

  • px: Absolute length in pixel
  • %: Relative to the value of the parent element
  • em: Relative to the font-size of the element
  • rem: Relative to the font-size of the root
  • vh: Equal to 1% of the height of the browser window
  • vw: Equal to 1% of the width of the browser window

Absolute vs Relative

Absolute is non-responsive whilst relative is responsive. Absolutes are px, in cm etc whilst relatives are %, em, rem, vh, vw, vmin, vmax etc.
